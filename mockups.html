Information

This was a class assignment done in our second Adobe Flash course and is (clearly) a remake of the classic Space Invaders arcade game. It's currently missing its preloader, and there are a few things I would like to tweak about the gameplay mechanics and graphics, but otherwise the game is both complete and playable.

I previously held contests to see who could get the highest score, but then I changed where the game was stored on my website and broke the scoreboard... Oops. It's an easy fix but I no longer own Flash.

Technologies

This game uses nothing more than ActionScript 2 and PHP/MySQL. The graphics were designed by-hand using the awesome Pixen graphics program for Mac OS X, and the music is the title track to "Future, and It Doesn't Work" by Infinity Shred (aka. Starscream).
