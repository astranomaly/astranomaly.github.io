Omniculus Systems Kiosk

This fine fictional house feature actually started as a simple UI design. The idea was to plan and design a home security system touchscreen kiosk. A year later, I added some simple interactivity by moving the design into Flash and connecting the program to an RGB LED light. The program has three sliders (RED, GREEN, BLUE) that increase or diminish the intensity of the LED. The kiosk also has a toggle ON/OFF button, which fully raises or lowers all three values.

This system uses ActionScript 3 and as3glue under the UI in order to connect to an Arduino platform through Serproxy. The Arduino then connects to an RGB LED hidden underneath a stylish ambient egg light. If that all seems like a lot of work, it's because it was.
