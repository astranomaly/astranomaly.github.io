Space Invaders || Astranomaly

This was a class assignment done in our second Flash course. It's an ActionScript 2 remaking of the classic Space Invaders arcade game. It's currently missing its preloader, and there are a few things I would like to tweak about the gameplay mechanics and graphics, but otherwise this game is complete and playable.

Occasionally I hold contests to see who can get the highest score. Prizes ensue. Though, there is no contest going on right now.

This game uses nothing more than ActionScript 2 and PHP/MySQL. The graphics were designed using the awesome Pixen graphics program for Mac OS X, and the music is the title track to Starscream's "Future, and It Doesn't Work".
